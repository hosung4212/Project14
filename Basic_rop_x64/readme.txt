이번거는 아까거 숙련인데 일단 새로운 명령어들이 있다. libc의 심볼을 가져왔고 ROP(e) 를 이용해 명령어로 가젯을 가져올 수 있었따. 그리고 PIE가 해제 되어있으므로 코드에 있는 main은 절대 주소를 가진다. e.symbols['main'] 은 실제 main의 주소를 가진다. 
라이브러리의 심볼을 저장하여 라이브러리에서의 상대 주소도 가져왔다.
그럼 read_got 값을 이용해 실제 read값을 구하고 라이브러리 실제 시작주소를 구한다면 라이브러리의 심볼을 이용해서 나머지 값들을 구할 수 있다.
마지막 출력도 0x40 으로 받아주면서 첫번째 main의 ret이 작동하게 했고 두번째 main 작동에서는 ret에 pop rdi 와 binsh 그리고 system 함수가 실행하게 만들어 /bin/sh 를 실행하게 만들었다.
